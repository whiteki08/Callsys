# 餐饮排队叫号系统

## 开发时间
2023.8.1-2023.8.7
## 项目简介
本人于2022年5月在~~广州市第二中学~~与同学组队完成了一项信息技术作业，但受限于开发时间和个人能力，只实现了基本的功能，现打算将其完善。
## 预期功能
### 原有功能
	1.使用pushplus服务实现了扫码取号，排队叫号
	2.在取号的同时告知用户当前排队人数及预计等待时间
### 新增功能
	1.优化扫码流程，增加扫码成功的检测
	2.增加实时查询当前人数的接口
	3.设计gui界面,优化配置流程
	4.引入ChatGPT
## 开发平台
MSVC,Python,OpenAI,pushplus
